Smoothing of values measured along river networks

Background

DEMs are subject to errors. There is some uncertainty about the elevation of each pixel. This is due to several reasons including data representation, artifacts, and random errors. These errors occur particularly along thalwegs in mountainous topography with its steep valley sidewalls. Longitudinal river profiles thus exhibit considerable scatter that overprints the actual patterns.
TopoToolbox includes several functions that allow smoothing errors in data measured along river networks (Schwanghart and Scherler, 2017). Elevation values are such data, but smoothing might also be useful with other data such as along-river gradients, or ksn values. This user guide presents the available functions and shows applications.

Functions for smoothing

Following functions are available for smoothing data measured along the river network
  • STREAMobj/crs
  • STREAMobj/crslin
  • STREAMobj/smooth
  • STREAMobj/crsapp
crs uses quantile regression, whereas crslin and smooth employ a least squares approach to model the mean conditional on the distance from the outlet. crs and crslin allow setting monotonicity constraints and are thus particularly useful for smoothing elevation values. The monotonicity constraint ensures that elevation values are decreasing in downstream direction. crsapp provides a visual interface to the crs function.Smoothing of all algorithms relies on curvature regularization. The amount of smoothing is controlled by the smoothing parameter K.

Smoothing the longitudinal river profile

DEM = GRIDobj('srtm_bigtujunga30m_utm11.tif');
FD = FLOWobj(DEM);
S = STREAMobj(FD,'minarea',1000);
S = klargestconncomps(S);
S = trunk(S);
z = getnal(S,DEM);
zs = crs(S,z,'split',0,'K',10,'tau',0.5);
plotdz(S,z,'Color',[.5 .5 .5])
hold on
plotdz(S,zs)
hold off

Visualizing the steepness of slopes adjacent to the river network

SE = STREAMobj(FD,'minarea',1000);
[D,ix] = drainagebasins(FD,SE.IXgrid);
I = D>0;
G = gradient8(DEM);
g = accumarray(ix(D.Z(I.Z(:))),G.Z(I.Z(:)),[numel(DEM.Z) 1],@mean,nan([1 1],'single'));
G.Z = reshape(g,DEM.size);
g = getnal(S,G);
plotdz(S,g)
hold on
gs = smooth(S,g,'k',100);
plotdz(S,gs,'color','k','LineWidth',2)
hold off
ylabel('Hillslope gradient [-]')
Finally, plot the smoothed gradient as color that varies along the longitudinal river profile
figure
plotdz(S,zs,'color',gs,'colormethod','surface')
box on
